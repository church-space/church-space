# Subprocessors

_Last updated: April 13, 2025_

<br />

We use third party subprocessors, such as cloud computing providers and customer support software, to run Church Space (the service). We extend our [GDPR safeguards](https://churchspace.co/policies/privacy) everywhere personal data is processed, including with third-party subprocessors.

The following is a list of personal data subprocessors we use. These subprocessors are all located in the United States:

- [Supabase](https://supabase.com/legal/dpa). Cloud services provider.
- [Resend](https://resend.com/legal/dpa). Transactional email service.
- [Superhuman](https://superhuman.com/dpa). Email software for support and sales.
- [Stripe](https://stripe.com/legal/dpa). Payment processing services.
- [Vercel](https://vercel.com/legal/dpa). Cloud services provider.
- [Open AI](https://openai.com/policies/data-processing-addendum/). AI requests.
- [Google Cloud](https://cloud.google.com/terms/data-processing-addendum). AI requests and business software.
- [Trigger.dev](https://trigger.dev/legal). Cloud services provider.
- [Upstash](https://upstash.com/static/trust/dpa.pdf). Cloud services provider.
- [Posthog](https://posthog.com/docs/privacy/gdpr-compliance). Analytics and AB testing.
- [Cloudflare](https://www.cloudflare.com/gdpr/introduction/). Cloud services provider.
